# Pimmy
Rew's package manager

## Commands
```bash
pimmy -Rs
pimmy -Aa package.name // app
pimmy -L // list akk
pimmy -Li // list installed
pimmy -As // sync app
pimmy -Ar // add repo
pimmy -Aq // search app
pimmy -Rq // search repo
pimmy -Ar // remove app
pimmy -Rr // remove repo

pimmy -Ta . // add types to the current app
pimmy -Ts . // sync types to the current app

pimmy -Am <path|name> // open readme

pimmy -Ma modname // pimmy mod add
pimmy -Ms modname // pimmy mod sync
pimmy -Mr modname // pimmy mod rm
pimmy -Mx qrew ./something.brew -o ./out.qrew // mod exec



```
